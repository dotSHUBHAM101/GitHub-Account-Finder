#  GitHub Finder

A modern React-based GitHub Finder application that allows users to search any GitHub profile and explore detailed account information, repositories, and repository links in a clean UI.

---

## 🌟 Features

### 🔍 Search GitHub Users
- Search any GitHub username
- Dynamic profile fetching using GitHub API

### 🕘 Search History
- Previous searches are stored using Local Storage
- Quickly revisit searched profiles
- Delete search history entries

### 👤 Profile Page
Displays:
- Profile Picture
- Name
- Bio
- Followers
- Following
- Repository Count
- Location
- GitHub ID

### 📂 Repositories Page
- Displays all public repositories
- Repository description
- Repository language
- Stars & forks count
- Direct repository URL button

### 🛣 React Router Integration
Implemented multi-page navigation using:
- `Routes`
- `Route`
- `useNavigate`
- `useParams`

### ⚡ Custom Hooks
Custom hooks are used for cleaner logic separation and reusable functionality.

### 💾 Local Storage
Used to:
- Save previous search history
- Improve user experience

### 🎨 Clean UI
- Responsive dark themed design
- Organized layout
- Modern card-based UI
